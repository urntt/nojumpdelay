# NoJumpDelay

A simple Fabric client-side mod that removes the player's jump cooldown.

## Features

* Removes jump delay
* Client-side only
* Lightweight and fast

## Requirements

* Minecraft 26.1
* Fabric Loader 0.18.5+
* Fabric API 0.144.3+

## Installation

1. Download the latest `.jar` from Releases
2. Put it into your `.minecraft/mods` folder
3. Launch the game

## Development

Build the mod:

```bash
./gradlew build
```

The jar will be in:

```path
build/libs/
```
